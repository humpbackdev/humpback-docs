# What's in the box?

As we mentioned before the main objective of this project is to improve the startup times of your projects giving you all the tools that you may need during the development process of your site.

- Humpback includes:
- PHP-FPM
- Nginx
- Mariadb
- Solr
- Selenium
- Ngrok
- Varnish
- MailHog
- Traefik
- Custom cli
